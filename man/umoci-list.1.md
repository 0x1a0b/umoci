# NAME
umoci list - List tags in an OCI image

# SYNOPSIS
**umoci list**
**--layout**=*image*

**umoci ls**
**--layout**=*image*

# DESCRIPTION
Gets the list of tags defined in an OCI image, with one tag name per line.

# OPTIONS

**--layout**=*image*
  The OCI image layout to get the list of tags from. *image* must be a path to
  a valid OCI image.
